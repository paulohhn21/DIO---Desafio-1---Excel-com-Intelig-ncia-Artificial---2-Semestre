# DIO---Desafio-1---Excel-com-Intelig-ncia-Artificial---2-Semestre
Desafio Proposto 1 - Ferramenta de simulação de investimentos

Ferramenta criada com base nas aulas do instrutor Felipe Aguiar sobre o tópico. 
Além de colocar em prática aquilo mostrado pelo instrutor, foi também adicionado um comparativo visual entre outras formas de investimento assim como a possibilidade de se inserir um aporte inicial para os investimentos, além dos aportes mensais. 
A ferramenta também demonstra um comparativo entre o valor bruto investido e o valor adquirido com dividendos, aumentando a gama de informações geradas a partir do uso do simulador
